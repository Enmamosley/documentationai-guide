## Requisitos

- Node 20+ y npm 10+
- Acceso al proyecto Supabase `wcg-app` (id `lvqqzpabnmdkamxgxrob`, región `us-east-2`)
- Las credenciales públicas (`NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY`) viven en `.env.example`

## Levantar el entorno

<Steps>
  <Step title="Clonar y configurar" icon="download" title-type="p">
    ```bash
    git clone https://github.com/Enmamosley/wcg-app.git
    cd wcg-app
    npm install
    cp .env.example .env.local
    ```
    El `.env.example` ya viene poblado con la URL y la publishable key del proyecto Supabase compartido.
  </Step>

  <Step title="Arrancar dev server" icon="zap" title-type="p">
    ```bash
    npm run dev
    ```
    Abre http://localhost:3000. El middleware te redirige a `/login`.
  </Step>

  <Step title="Login con la cuenta demo" icon="lock" title-type="p">
    - **Email**: `admin@splanding.dev`
    - **Password**: `wcg-admin-2026`
    - **Rol**: admin (ve todo)

    Para probar otros roles, crea usuarios desde `/sistema` → Usuarios.
  </Step>

  <Step title="Pasea por las pantallas" icon="navigation" title-type="p">
    El sidebar muestra **9 módulos**. Usa **⌘K / Ctrl+K** para abrir el command palette y buscar folios, alfanuméricos, facturas o RFCs.
  </Step>
</Steps>

## Genera tu primer F200

<Steps>
  <Step title="Entra a /f200" icon="download" title-type="p">
    Selecciona periodo (ej. `2026-05-W19`), técnico y tipo de servicio.
  </Step>
  <Step title="Verifica el preview" icon="eye" title-type="p">
    El preview tipo spreadsheet muestra exactamente las filas que se exportarán. Si hay OS no liquidadas, no aparecen — eso es esperado.
  </Step>
  <Step title="Descarga el .xlsx" icon="download" title-type="p">
    El botón llama a `GET /api/f200` con los filtros como query params. El navegador descarga `F200_2026-05-W19.xlsx`.
  </Step>
</Steps>

<Callout kind="tip">
  El F200 sale con el formato exacto del template Telmex v2.4: header congelado, números a la derecha con `#,##0.00`, totales al pie. Súbelo directo al portal sin pegar nada en Excel.
</Callout>

## Scripts útiles

```bash
npm run dev      # dev server con Turbopack
npm run build    # build de producción
npm run lint     # eslint

# Re-seed reproducible (rara vez se corre)
node scripts/build-seed.mjs > scripts/seed.sql
node scripts/build-telmex-import.mjs > scripts/telmex-import.sql

# Regenerar tipos desde Supabase
npx supabase gen types typescript --project-id lvqqzpabnmdkamxgxrob > src/lib/database.types.ts
```
